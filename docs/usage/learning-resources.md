# 学习资源推荐

- **能力短板定位**：识别薄弱环节，针对性推荐资源
- **动态更新**：每次写作后，推荐内容实时调整

- **语法讲解视频**：针对高频语法错误，推荐权威讲解视频
- **词汇拓展练习**：根据词汇使用情况，推送同义词、短语等练习

- **专项训练题库**：针对薄弱点，生成个性化训练题
